Experiment 1: Introduction to Python

Objective

Install Python and set up the development environment.

Tasks

1. Install Python and an IDE

Step 1: Download Python

  • Go to the official Python website.
  • Download the latest version of Python for your operating system (Windows, macOS, or Linux).
  • Make sure to check the box that says "Add Python to PATH" during installation.

Step 2: Install an IDE

Choose one of the following:

  • PyCharm
    • Download from JetBrains PyCharm.
    • Install the Community edition (free).
  • VSCode
    • Download from Visual Studio Code.
    • Follow the installation instructions, and consider installing the Python extension from the Extensions Marketplace.
  • Jupyter Notebook
    • If you prefer Jupyter, you can install it via Anaconda (Anaconda Distribution) or directly using pip:
      • pip install notebook

2. Write and Run a Simple "Hello, World!" Program

Using your chosen IDE:

  • Open your IDE.
  • Create a new Python file (e.g., hello.py).
  • Write the following code:
    • print("Hello, World!")
  • Save the file and run the program.
    • In PyCharm: Right-click the file and select "Run".
    • In VSCode: Open the terminal and type python hello.py.
    • In Jupyter Notebook: Create a new notebook cell and run the code.

3. Understand and Demonstrate Basic Python Syntax and Semantics

Basic Concepts to Explore:

  • Variables and Data Types 
    name = "Alice"  # String
age = 25        # Integer
height = 5.5    # Float
is_student = True  # Boolean
  • Basic Operations 
    a = 10
b = 5
sum_result = a + b  # Addition
  • Control Structures 
    # If statement
if age >= 18:
    print("Adult")
else:
    print("Minor")

# For loop
for i in range(5):
    print(i)
  • Functions 
    def greet(person):
    return f"Hello, {person}!"

print(greet("Alice"))

Conclusion

You've successfully installed Python and an IDE, run a basic program, and explored some fundamental concepts of Python programming. This sets a solid foundation for further experimentation and learning in Python!
